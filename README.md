# Makergram - IN PROGRESS!

A Instagram-clone written in Rails

* Ruby version: 2.3.0
* Rails version: 5.0.0.1

## User stories

```
As a User
After taking a photo I'm proud of
I want to be able to post the photo in my Makergram account
```
```
As a User
To keep up to date with my friends
I want to be able to view photos they've posted
```
```
As a User
So I can state my opinions on other people's photos
I want to be able to leave a comment on their photos
```
```
As a User
So I can revisit photos I've taken
I want to be able to sign up for a Makergram account and log in/out as I please
```
```
As a User
So I can choose only my finest photos to be available
I want to be able to remove photos I've posted
```
```
As a User
So I can show my appreciation for others' photos
I want to be able to leave an emoji reaction on their photos
```
```
As a User
So I can prove how popular I am
I want to be able to view a count of the number of reactions my photos have received
```
```
As a User
So I can retract views I've previously stated
I want to be able to remove comments I've posted
```


## Setup

* Fork this repo
* Install gems: run 'bundle install'
* Setup your databases: run 'rails db:migrate'
* Run the test suite: run 'rspec'
* Boot up local version: run 'rails server'
